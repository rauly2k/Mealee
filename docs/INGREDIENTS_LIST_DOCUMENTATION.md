# Standard Ingredients List Documentation

This document defines the standardized list of ingredients used in the Mealee app. Standardizing ingredients improves search functionality, recipe matching, and shopping list management.

## Purpose

Having a standardized ingredients list helps with:
1. **Consistent naming** - Same ingredient always has same name
2. **Better search** - Users can find recipes by ingredients more easily
3. **Shopping list aggregation** - Same items combine properly
4. **Nutritional accuracy** - Standardized nutrition data
5. **Pantry management** - Track what you have accurately

## Implementation Guidelines

### Naming Conventions

1. **Use lowercase** - All ingredient names in lowercase
2. **Singular form** - Use singular except where plural is standard
3. **Be specific** - "piept de pui" not just "pui"
4. **Romanian first** - Primary language is Romanian

### Handling Variations

When an ingredient has multiple names:
- Choose the most common Romanian name
- Add aliases in the database for search
- Example: "roșii" (primary) with alias "tomate"

### Adding New Ingredients

When adding ingredients not on this list:
1. Check if it can be categorized under existing ingredients
2. Use the standard naming format
3. Add to appropriate category
4. Update this documentation
